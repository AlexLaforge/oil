## Installation

```
npm install
```



## Usage

### Basic Usage

* Start App locally: `npm start`
* Running unit-tests: `npm run test`
* Running end2end-tests locally: Start app in one terminal `npm start` and then in the next terminal `npm run e2e`

### Advanced Usage

### debug logging

NODE_DEBUG=oil-debug npm run ...
